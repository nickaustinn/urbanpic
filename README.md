# UrbanPic

AI-powered 311 civic issue reporter. Snap a photo, let Claude classify it, and automatically route a report to the right city department.

## Team
- [@nickaustinn](https://github.com/nickaustinn)
- [@mollindabo](https://github.com/mollindabo)

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Next.js 14, Tailwind CSS |
| AI | Claude Vision (Anthropic) |
| Database | PostgreSQL + Prisma ORM |
| Maps | Mapbox GL JS |
| Auth | JWT (stretch) |
| Deploy | Vercel + Supabase |

## Features

- **Photo capture** — upload or take a photo of any city issue
- **AI classification** — Claude Vision identifies issue type and severity
- **Auto-routing** — reports sent to correct city department automatically
- **311 report generation** — structured report description generated by Claude
- **Community heat map** — Mapbox visualization of all submitted reports
- **User accounts** *(stretch)* — track your submissions and their status

## Getting Started

```bash
# 1. Clone
git clone https://github.com/nickaustinn/urbanpic.git
cd urbanpic

# 2. Install dependencies
npm install

# 3. Set up environment
cp .env.example .env.local
# Fill in your API keys

# 4. Set up database
npx prisma migrate dev

# 5. Run dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Environment Variables

See `.env.example` for required variables:
- `ANTHROPIC_API_KEY` — Claude API key
- `NEXT_PUBLIC_MAPS_API_KEY` — Mapbox public token
- `DATABASE_URL` — PostgreSQL connection string
- `JWT_SECRET` — Secret for JWT signing
